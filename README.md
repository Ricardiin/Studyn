# 📚 StudyIn — Sistema Inteligente de Estudos

## 🚀 Sobre o Projeto

O **StudyIn** é uma plataforma web desenvolvida para ajudar estudantes a organizarem seus estudos de forma inteligente, visual e gamificada.

O sistema une:

- 📖 Leitura
- ✅ Gerenciamento de tarefas
- ⏱️ Controle de tempo
- 🎮 Gamificação
- 📊 Evolução visual
- 🚀 Produtividade

Tudo isso em uma experiência moderna e motivadora.

---

- Java
- JSP
- Servlet
- MySQL
- HTML5
- CSS3
- JavaScript
- JDBC
- Apache Tomcat

# 🔐 1. Sistema de Cadastro/Login

## Funcionalidades

- ✅ Cadastro de usuários
- ✅ Login com autenticação
- ✅ Sessão de usuário
- ✅ Logout
- ✅ Perfil personalizado
- ✅ Upload de foto de perfil usando BLOB no MySQL

---

# 📚 2. Biblioteca / Leitura

Sistema de gerenciamento de livros e leitura.

## Funcionalidades

- ✅ Adicionar livros
- ✅ Lista de leitura
- ✅ Editar livros
- ✅ Excluir livros

## Status de leitura

- 📖 Lendo
- ✅ Concluído
- ⭐ Quero ler

## Progresso

- 📊 Barra de progresso da leitura
- 📈 Percentual concluído (%)

## 💡 Diferencial

Sistema de metas:

- 📄 10 páginas por dia
- 📅 50 páginas por mês

---

# ✅ 3. Sistema de Tarefas (To-Do List)

Gerenciamento completo de tarefas.

## Funcionalidades

- ✅ Criar tarefas

- ✅ Excluir tarefas

- ✅ Concluídas

# 🏗️ 7. Sistema de Construção
## 🌆 Cidade do Conhecimento

Um sistema visual onde cada atividade constrói parte da cidade do usuário.

---

## 🧩 Cada atividade gera construções

| Atividade | Construção |
|---|---|
| 📚 1h estudada | Biblioteca |
| ✅ Tarefas concluídas | Casas |
| 🔁 Consistência | Monumentos |
| 📖 Livros finalizados | Torres |
| 🎯 Metas concluídas | Estruturas especiais |

---

## 💡 Diferencial Principal

Cada tipo de estudo gera uma evolução visual única.

Isso transforma produtividade em:

- 🚀 Progressão
- 🏆 Coleção
- 💡 Motivação
- 🎨 Personalização

Inspirado em sistemas de jogos e cidades evolutivas.
